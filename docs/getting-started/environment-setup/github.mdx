---
title: Connecting to GitHub
---

To use the GitHub integration, you’ll need an existing GitHub repository or create one before connecting.

<Steps>
  <Step title="Go to Account Settings">
    In the Globe dashboard, go to Account → Connected Accounts
  </Step>
  <Step title="Connect Your GitHub Account">
    Select Connect your GitHub account
  </Step>
  <Step title="Authorize Dart Globe">
    Authorise access to your GitHub repositories
  </Step>
</Steps>

You can now deploy projects from your repos using github events triggers.
